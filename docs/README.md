## 文档体系

本项目采用分层文档体系，覆盖需求、开发、测试、运维全流程。

```
docs/
├── README.md                 # 文档导航（本文档）
│
├── requirements/            # 需求文档
│   ├── PRD-零代码微服务平台.md    # 产品需求文档
│   └── feature-list.md           # 功能清单
│
├── architecture/             # 架构文档
│   ├── 技术架构设计.md           # 技术架构设计
│   ├── 数据库设计.md             # 数据库设计
│   └── 接口设计.md               # 接口设计
│
├── function/                 # 功能文档
│   └── 功能说明文档.md           # 功能详细说明
│
├── iteration/                # 迭代文档
│   └── 迭代计划.md              # 版本迭代计划
│
└── standards/               # 规范文档
    ├── 编码规范.md              # Java编码规范
    └── Git规范.md               # Git工作流规范
```
