## 一、数据库规划

### 1.1 数据库列表

| 数据库名 | 服务 | 字符集 | 说明 |
|----------|------|--------|------|
| ry-cloud | system/auth | utf8mb4 | 系统核心库 |
| ry-gen | gen | utf8mb4 | 代码生成库 |
| ry-workflow | workflow | utf8mb4 | 工作流库 |
| ry-nocode | nocode | utf8mb4 | 零代码配置库 |

### 1.2 命名规范

| 规则 | 规范 | 示例 |
|------|------|------|
| 表名 | 小写字母 + 下划线 | sys_user |
| 字段名 | 小写字母 + 下划线 | user_name |
| 索引名 | idx_ + 表名 + 字段 | idx_sys_user_name |
| 主键名 | pk_ + 表名 | pk_sys_user |
| 外键名 | fk_ + 表名_关联表 | fk_sys_role_user |

│   sys_dept  │       │   sys_user  │       │   sys_role  │
├─────────────┤       ├─────────────┤       ├─────────────┤
│ dept_id(PK) │◀─────│ dept_id(FK) │       │ role_id(PK) │
│ parent_id   │       │ user_id(PK) │       │ role_key    │
│ dept_name   │       │ user_name   │       │ role_name   │
└─────────────┘       └──────┬──────┘       └──────┬──────┘
                            │                    │
                            │                    │
                    ┌───────┴───────┐      ┌──────┴──────┐
                    │sys_user_role │      │ sys_role_menu│
                    ├──────────────┤      ├─────────────┤
                    │user_id(FK)   │      │ role_id(FK)  │
                    │role_id(FK)   │      │ menu_id(FK)  │
                    └──────────────┘      └──────────────┘
                                               │
                                               ▼
                                    ┌─────────────┐
                                    │  sys_menu   │
                                    ├─────────────┤
                                    │ menu_id(PK) │

                                    │ menu_name  │

## 四、索引设计

### 4.1 索引规范

| 索引类型 | 使用场景 |
|---------|---------|
| 主键索引 | 每表必须有，InnoDB聚集索引 |
| 唯一索引 | 业务唯一字段（手机号、邮箱） |
| 普通索引 | WHERE/JOIN/ORDER BY 字段 |
| 复合索引 | 多字段联合查询，遵循最左前缀 |

### 4.2 核心索引示例

```sql
-- 用户表索引
ALTER TABLE sys_user ADD UNIQUE INDEX idx_user_name (user_name);
ALTER TABLE sys_user ADD INDEX idx_dept_id (dept_id);
ALTER TABLE sys_user ADD INDEX idx_status (status);

-- 角色表索引
ALTER TABLE sys_role ADD UNIQUE INDEX idx_role_key (role_key);
ALTER TABLE sys_role ADD INDEX idx_status (status);

-- 菜单表索引
ALTER TABLE sys_menu ADD INDEX idx_parent_id (parent_id);
ALTER TABLE sys_menu ADD INDEX idx_status (status);
```
