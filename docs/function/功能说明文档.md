## 一、插件热插拔引擎

### 1.1 功能概述

插件热插拔引擎是本平台的核心能力之一，基于PF4J实现业务模块的动态加载、卸载、启用、停用，无需重启应用即可上线新功能。

### 1.2 功能详情

#### 1.2.1 插件安装

| 项目 | 说明 |
|------|------|
| 功能描述 | 支持JAR包本地安装和Maven坐标远程安装 |
| 输入 | 插件JAR路径或Maven坐标 |
| 处理流程 | 1. 解析插件元数据<br>2. 下载依赖（如需要）<br>3. 加载插件类<br>4. 初始化Spring Bean |
| 输出 | 插件就绪状态 |
| 异常处理 | 依赖冲突、类冲突、版本不兼容 |

#### 1.2.2 插件卸载

| 项目 | 说明 |
|------|------|
| 功能描述 | 完全卸载插件，清理所有资源 |
| 输入 | 插件ID |
| 处理流程 | 1. 停止插件运行<br>2. 销毁Spring Bean<br>3. 卸载类加载器<br>4. 清理临时文件 |
| 输出 | 卸载成功状态 |

#### 1.2.3 插件启用/停用

| 项目 | 说明 |
|------|------|
| 功能描述 | 启用：注册路由/API/初始化数据<br>停用：注销路由/API/保留配置 |
| 输入 | 插件ID |
| 输出 | 启用/停用状态 |

#### 1.2.4 插件隔离

| 项目 | 说明 |
|------|------|
| 功能描述 | 每个插件拥有独立的类加载器，防止类冲突 |
| 实现方式 | PF4J PluginClassLoader |
| 隔离内容 | - 插件自身类<br>- 第三方依赖（可选） |

### 1.3 插件开发规范

```java
// 插件入口类示例
public class MyPlugin extends Plugin {
    
    @Override
    public void start() {
        // 插件启动时的初始化逻辑
    }
    
    @Override
    public void stop() {
        // 插件停止时的清理逻辑
    }
}

// 插件定义文件 (plugin.properties)
plugin.id=com.example:my-plugin
plugin.class=com.example.plugin.MyPlugin
plugin.version=1.0.0
plugin.provider=Example
plugin.description=我的插件
plugin.dependencies=
```

## 二、代码生成引擎

### 2.1 功能概述

代码生成引擎实现从数据库表到完整代码的一键生成，包括后端Controller/Service/Mapper/Entity和前端Vue页面。

### 2.2 功能详情

#### 2.2.1 表结构解析

| 项目 | 说明 |
|------|------|
| 功能描述 | 连接数据库读取表结构信息 |
| 输入 | 数据库连接信息 + 表名 |
| 输出 | 表名、字段列表、字段类型、注释、索引、约束 |

#### 2.2.2 模板渲染

| 项目 | 说明 |
|------|------|
| 功能描述 | 使用Velocity模板引擎渲染代码 |
| 模板类型 | Controller、Service、ServiceImpl、Mapper、MapperXml、Entity |
| 前端模板 | Vue列表页、Vue表单页、Vue类型定义 |

#### 2.2.3 代码生成

| 输出类型 | 文件 |
|----------|------|
| 后端 | Entity.java、Controller.java、Service.java、ServiceImpl.java、Mapper.java、Mapper.xml |
| 前端 | index.vue、form.vue、types.ts、api.ts |

### 2.3 生成配置

```yaml
# 代码生成配置
gen:
  # 包配置
  package:
    packageName: com.ruoyi.nocode
    moduleName: business
    businessName: customer
    
  # 作者
  author: AI
  autoRemovePre: false
  tablePrefix: sys_
  
  # 字段配置
  column:
    - columnName: id
      columnType: Long
      javaType: Long
      isPk: true
    - columnName: name
      columnType: varchar
      javaType: String
      isRequired: true
```

## 四、Json2UI动态表单

### 4.1 功能概述

Json2UI是前端表单渲染的核心引擎，通过JSON Schema动态渲染Vue组件，支持拖拽设计、AI生成、事件绑定、数据源绑定。

### 4.2 功能详情

#### 4.2.1 表单渲染

```json
{
  "formId": "customer_form",
  "formName": "客户信息表单",
  "components": [
    {
      "type": "input",
      "field": "customerName",
      "label": "客户姓名",
      "required": true
    },
    {
      "type": "select",
      "field": "customerType",
      "label": "客户类型",
      "dataSource": {
        "type": "api",
        "url": "/api/dict/customer_type"
      }
    }
  ]
}
```

#### 4.2.2 事件绑定

```javascript
// 客户类型变更时触发
onCustomerTypeChange: `
  if (this.customerType === 'VIP') {
    this.showVipFields = true;
  }
`
```

#### 4.2.3 数据源绑定

| 类型 | 说明 |
|------|------|
| API数据源 | HTTP请求获取数据 |
| 字典数据源 | 系统字典表 |
| 静态数据源 | 固定选项列表 |
| 表达式数据源 | JS表达式计算 |
